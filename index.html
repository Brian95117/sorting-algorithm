Bubble Sort

透過相鄰元素比較與交換，讓較大的值逐步移到右邊。

O(n²) 點擊進入
🎯

Selection Sort

每一輪從未排序區間中挑出最小值，放到前方正確位置。

O(n²) 點擊進入
📥

Insertion Sort

把元素插入已排序區間中，很像整理撲克牌的方式。

O(n²) 點擊進入
🧩

Merge Sort

先分割資料，再合併排序，屬於分治法中的經典演算法。

O(n log n) 點擊進入
⚡

Quick Sort

利用 pivot 進行分區，再對左右兩側遞迴排序。

O(n log n) 點擊進入
