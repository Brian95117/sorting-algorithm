Bubble Sort

透過相鄰元素兩兩比較與交換，讓較大的值逐步移到右側。

O(n²) 相鄰交換
🎯

Selection Sort

每一輪從未排序區間中找出最小值，放到前方正確位置。

O(n²) 選最小值
📥

Insertion Sort

把新元素插入已排序區間中，就像整理撲克牌一樣。

O(n²) 插入排序
🧩

Merge Sort

先分割再合併，屬於經典分治法，效率比平方級排序更佳。

O(n log n) 分割合併
⚡

Quick Sort

選擇 pivot 分區，再分別對左右兩側遞迴排序。

O(n log n) Pivot 分區
